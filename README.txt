Student Enrollment Management System

Purpose:

	The Student Enrollment Management System aims to simplify the student enrollment process for high schools. It provides a user-friendly interface for administrators to input and manage student information, ensuring secure storage and retrieval with SQLite integration. With features for viewing and deleting records, the system supports efficient enrollment management, enhancing administrative workflows and promoting effective school operations.

Installation:

	Start by cloning the project repository from GitHub to your local machine using Git. Once cloned, navigate to the project directory using your terminal or command prompt. Ensure you have Python installed on your system and then install the required dependencies by running 'pip install -r requirements.txt.'  With the dependencies installed, launch the application by executing the main Python script with the command python main.py. 

Dependencies:

	The application requires the following dependencies:
		tkinter: Python's standard GUI library.
		sqlite3: SQLite database integration.
		ttk: Tkinter themed widgets.

Usage:

	1. Clone the Project:: Begin by cloning the project repository from GitHub to your local machine using Git.

	2. Navigate to the Project Directory:: Use your terminal or command prompt to navigate to the directory where the project is located.
	
	3. Install Dependencies:: Ensure Python is installed on your system, then install the required dependencies by running `pip install -r requirements.txt`.

	4. Launch the Application:: Execute the main Python script by running `python main.py` in your terminal or command prompt to start the application.

	5. Input Student Information:: Once the application window appears, input student details such as name, previous school, contact information, grade level, and preferred strand into the respective fields.

	6. Save Student Details:: Click the "Take Input" button to save the entered student details into the system.

	7. View Enrolled Students:: To view all enrolled students, click the "Display Result" button, which opens a new window displaying a list of student records.

	8. Manage Student Records:: From the result window, you can delete specific student records as needed by selecting them and clicking the "Delete" button.

	9. Efficient Enrollment Management:: With these features, the Student Enrollment Management System simplifies the student enrollment process, providing an intuitive interface for administrators to efficiently manage enrollment data.
